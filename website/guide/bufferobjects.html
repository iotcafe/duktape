Buffer objects

Overview of buffer types

Duktape provides the following buffer and buffer-related types:

  • Plain buffer value
  • Duktape.Buffer object
  • Node.js Buffer object
  • ArrayBuffer object
  • DataView and TypedArray views

Buffers in C code

Duktape API calls prior to 1.3 deal with plain buffer values only. To avoid incompatible changes in Duktape 1.3, separate API calls have been added to deal with any buffer-like values, including both plain buffers and buffer objects. API calls dealing with buffer objects have been tagged bufferobject in the API documentation:
